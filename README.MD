# 환율 계산

### Tech

* Spring Boot 2.3.8
* Spring Web 5.2.12
* JUnit 4
* Thymeleaf 3.0.12
* jQuery 3.3.1
* Bootstrap 4.3.1

### Guides
* 환율은 페이지 접근 시 실시간으로 API를 호출하여 가져옵니다.
    * 수취국가 select 안에 넣어두고 수취국가를 변경할 때마다 이를 자바스크립트로 보여줍니다.
    * 무료 API는 월 250회 제한이 있습니다.
* 결과는 하단에 표시합니다.
